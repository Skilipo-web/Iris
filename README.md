# 👁️ Iris

Iris est une solution web moderne et légère conçue pour l'optimisation des interfaces utilisateur et la gestion fluide des données. Développé par **Skilipo-web**, ce projet vise à offrir une expérience intuitive, performante et facilement adaptable.

---

## ✨ Fonctionnalités
*   **Interface Responsive :** S'adapte à tous les types d'écrans (Mobile, Tablette, Desktop).
*   **Performance Optimisée :** Code léger pour un temps de chargement minimal.
*   **Design Épuré :** Focus sur l'essentiel pour une meilleure ergonomie.

## 🛠️ Stack Technique
*   **Frontend :** HTML5 / CSS3 / JavaScript (ES6+)
*   **Outils :** Git & GitHub pour le versioning

## 🚀 Installation rapide

1. **Cloner le projet**
   ```bash
   git clone https://github.com
   ```
2. **Lancer l'application**
   Ouvrez simplement le fichier `index.html` dans votre navigateur préféré.
